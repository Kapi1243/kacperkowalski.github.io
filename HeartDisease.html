Heart Disease Prediction AI Model

About the Project

Technologies Used: Python, Pandas, Scikit-Learn, Decision Trees, Data Visualization (Matplotlib, Seaborn)

The Heart Disease Model is a Big data uni project designed to predict the likelihood of heart disease in individuals based on various health-related features. Using a dataset containing patient information, such as age, cholesterol levels, blood pressure, and family history, I trained a classification model to identify patterns and predict the risk of heart disease. After pre-processing the data, including handling missing values and scaling features, I used algorithms like Decision Trees and Logistic Regression to build an accurate model. The project focused on not only achieving high accuracy but also on providing explainable results, allowing healthcare professionals to interpret the model’s predictions. The final model can be used to assess an individual’s risk, providing valuable insights for early diagnosis and prevention. This project showcases the potential of machine learning in healthcare, where predictive models can assist in saving lives by offering timely interventions.
