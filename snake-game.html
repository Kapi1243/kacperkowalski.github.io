Reinforcement Learning Snake Game

About the Project

Technologies: Python, Reinforcement Learning, Neural Networks

This project is an AI-powered version of the classic Snake game. The AI uses reinforcement learning to optimize its movement strategy, learning over time how to survive longer and maximize its score. By implementing Deep Q-Learning (DQN), the model continuously improves its decision-making, showcasing the power of machine learning in gameplay optimization.

This AI-powered Snake game uses Deep Q-Learning to optimize movement and maximize survival.
