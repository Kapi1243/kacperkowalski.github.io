Project Overview:

This project demonstrates the development of a high-fidelity Python simulation framework for turbojet engines, implementing comprehensive thermodynamic modeling with real-time parameter optimization and interactive visualization. The simulator incorporates the Brayton cycle with real gas effects, variable specific heats, compressible flow dynamics, and component efficiency modeling. The architecture follows clean code principles with full separation of concerns, type safety through comprehensive annotations, and extensive documentation. The project includes a command-line interface for batch processing, an interactive PyQt6 GUI for real-time parameter exploration, and advanced visualization capabilities for performance analysis. Model validation was performed against NASA CEA (Chemical Equilibrium with Applications), GasTurb performance data, and published engine specifications from major manufacturers, ensuring simulation accuracy and reliability.
